### 使用说明

在clone下本仓库后，使用**npm install**下载该项目所需的全部依赖（没有npm包管理工具的可以先去下node.js）。下载完毕后，在项目目录下（能看见src,public等目录的那个目录）输入**npm start**即可运行。
